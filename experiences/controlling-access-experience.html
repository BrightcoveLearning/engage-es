Controlar el acceso a una experiencia

En este tema, aprenderá cómo restringir el acceso a una experiencia utilizando perfiles de control de acceso.

Los perfiles de control de acceso se utilizan para controlar el acceso a las experiencias y se crean como parte de la configuración del proyecto. Una vez que se han creado los perfiles, se pueden asignar a experiencias para controlar el acceso. Para obtener información sobre la creación de perfiles de control de acceso, consulte Creación de perfiles de control de acceso.

Para asignar un perfil de control de acceso a una experiencia, siga estos pasos:

  1. Edita la experiencia.
  2. Hacer clic CONFIGURACIÓN DEL SITIO> Control de acceso en la navegación de la izquierda.
  3. Seleccione un perfil de control de acceso para la experiencia.
  4. Haz clic en Guardar.

Asegúrese de volver a publicar la experiencia después de realizar cualquier cambio en la configuración de Control de acceso.
