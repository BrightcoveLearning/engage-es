Uso del editor de experiencias para personalizar una experiencia web

En este tema, aprenderá a utilizar el Editor de experiencias para personalizar la apariencia de una experiencia web.

El Editor de experiencias se utiliza para personalizar la apariencia de las experiencias creadas con Brightcove Engage™. Haz clic en el título de la experiencia para abrir el Editor de experiencias. Con las experiencias web, el editor de experiencias ofrece la capacidad de:

  • Agregar objetos a las páginas
  • Agregar videos a un sitio
  • Personaliza el estilo del sitio
  • Controlar el acceso al sitio
  • Configurar los detalles del sitio

Para acceder a estas opciones, haga clic en los menús de la navegación izquierda.

Agregar objetos a una página

Se pueden agregar objetos como texto, imágenes y URL a las páginas, lo que le permite personalizar y personalizar su experiencia web. Para aprender a agregar objetos a las páginas, consulte Adición de imágenes, texto y enlaces a páginas de experiencia web.

Agregar videos a una experiencia web

Para agregar videos a una experiencia, haga clic en VIDEOS en la navegación de la izquierda. Para obtener detalles sobre cómo agregar videos a una experiencia web, consulte Agregar videos a una experiencia.

Personalización del estilo

Para personalizar los colores y fuentes utilizados en una experiencia web, o para seleccionar otro tema, haga clic en ESTILO en la navegación de la izquierda. Para obtener detalles sobre cómo personalizar el estilo, consulte Personalización del estilo de una experiencia web.

Controlar el acceso al sitio

La configuración de Control de acceso se utiliza para restringir el acceso al sitio. Para configurar los ajustes de Control de acceso, haga clic en CONTROL DE ACCESO en la navegación de la izquierda.

Para obtener detalles sobre la configuración del control de acceso, consulte Controlar el acceso a una experiencia.

Configurar los detalles de la experiencia

La configuración de Descripción general ofrece la posibilidad de editar el nombre, la descripción y el idioma de la experiencia. Para configurar los detalles de la experiencia, haga clic en VISIÓN DE CONJUNTO en la navegación de la izquierda.

Para obtener detalles sobre la configuración de los detalles del sitio, consulte Editar detalles de la experiencia.
