Asignar un dominio personalizado a una experiencia web

En este tema, aprenderá a asignar un dominio personalizado a una experiencia web.

Asignar un dominio personalizado a una experiencia web proporciona una forma de acceder al sitio en una URL fácil de recordar (en lugar de tener una site-slug.bcvp0rtal.com URL). Al usar dominios personalizados, también puede asignar una ruta opcional a la URL. El uso de rutas le permite utilizar diferentes plantillas, crear micrositios, campañas, portales internos, etc., sin dejar de usar el mismo dominio. Por ejemplo, videos.mycustomdomain.com se puede expandir para incluir también rutas como videos.mycustomdomain.com/event o videos.mycustomdomain.com/product. Las rutas permiten que los editores que tienen varios sitios y SSL solo tengan que configurar SSL para un dominio en lugar de cada slug de sitio individual.

Nota: Los dominios personalizados deben crearse como parte de la configuración del proyecto antes de pueden asignarse a una experiencia. Para obtener información, consulte Configuración de dominios personalizados y SSL.

Para asignar un dominio personalizado a una experiencia web, siga estos pasos:

  1. Edita la experiencia.
  2. Hacer clic CONFIGURACIÓN DEL SITIO> URL en la navegación de la izquierda. Se mostrará el dominio inicial asignado al sitio.
  3. (Opcional) Desmarque la ¿Activo? casilla de verificación para deshabilitar site-xxxx.bcvp0rtal.com domain. Normalmente, los editores inhabilitarán site-xxxx.bcvp0rtal.com cuando se utiliza un dominio personalizado para que los motores de búsqueda no lo indexen.
  4. Hacer clic Agregar dominio personalizado.
  5. Por defecto, los dominios serán Activo. Desmarcando Activo desactiva Brightcove Engage para que no se enrute en la entrada, pero no libera la combinación de ruta / dominio para su uso en otro sitio.
  6. Seleccione un dominio de la lista desplegable. Se mostrará la lista de dominios personalizados que se configuraron como parte de la Configuración del proyecto.
  7. Opcionalmente, ingrese una ruta para el dominio.
  8. Agregue dominios adicionales para el sitio haciendo clic en Agregar dominio personalizado.
  9. Cuando termine, haga clic en Ahorrar.

Haz clic en el icono de borrar () para eliminar un dominio personalizado.

Nota: Para que cualquiera de los cambios de dominio (incluida la activación o eliminación de un dominio) surta efecto, el sitio debe publicarse.

La lista de dominios personalizados aparecerá en el cuadro de diálogo de publicación.

Para deshabilitar o eliminar cualquiera de los dominios personalizados, edite el sitio, haga clic en CONFIGURACIÓN DEL SITIO> URL y desmarque la Activo casilla de verificación para desactivar el dominio o haga clic en el icono de eliminar () para eliminar el dominio personalizado.
