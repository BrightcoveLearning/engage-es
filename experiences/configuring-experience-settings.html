Configurar los ajustes de la experiencia

En este tema, aprenderá a configurar la experiencia.

Para configurar los ajustes de Experiencia, abra el Proyectos página y haga clic en el Ajustes enlace en la parte superior de la página. Se pueden configurar los siguientes ajustes:

  • Perfiles de control de acceso

Configuración de perfiles de control de acceso

Se pueden crear perfiles de control de acceso para controlar el acceso a las experiencias publicadas. El acceso a un sitio puede estar limitado por:

  • Credenciales SSO - Se requiere un nombre de usuario y una contraseña para acceder

Para obtener información sobre la creación de perfiles de control de acceso, consulte Creación de perfiles de control de acceso.
