Descripción general: Creación, edición y publicación de experiencias de aplicaciones móviles

En este tema, aprenderá los pasos básicos para crear, editar y publicar una experiencia de aplicación móvil.

Brightcove Engage™ le permite crear experiencias de vídeo atractivas sin necesidad de recursos técnicos. El siguiente es un ejemplo de una experiencia de aplicación móvil que se creó utilizando la plantilla de aplicación móvil Brightcove Engage.

Para obtener una guía paso a paso para crear una experiencia de aplicación móvil, consulte Paso a paso: Creación y publicación de una experiencia de aplicación móvil.

Los pasos principales para crear y publicar una experiencia de aplicación móvil son:

  • Elige una plantilla
  • Añadir vídeo y personalizar la experiencia
  • Vista previa y publicación de la experiencia

Elección de una plantilla

Cuando creas una nueva experiencia, empiezas por elegir una plantilla. La plantilla controla el aspecto general de la experiencia. Para obtener más información sobre las plantillas y ver algunos sitios de muestra, consulte la Descripción general de las plantillas de experiencia tema.

Después de elegir una plantilla, tiene la opción de rellenar previamente la experiencia con contenido de muestra. Haciendo clic Sí, rellenar previamente mi experiencia completará previamente la experiencia con videos de su cuenta de Brightcove. Esto hace que sea más fácil ver cómo será la experiencia al iniciar el proceso de personalización.

Adición de vídeo y personalización de la experiencia

Después de crear una experiencia, el Editor de experiencias se utiliza para personalizar diferentes aspectos de la experiencia. Se pueden realizar los siguientes tipos de personalizaciones:

  • Paginas - Se puede agregar un logotipo a la página de inicio para marcar la experiencia según sea necesario. Para obtener más información sobre cómo agregar un logotipo, consulte Agregar un logotipo a una experiencia de aplicación móvil.
  • Vídeos: las colecciones se crean para seleccionar y organizar el contenido de vídeo. Para obtener más información sobre cómo agregar contenido de video a la experiencia, consulte Agregar videos a una experiencia.
  • Estilo - El aspecto de la experiencia se puede personalizar seleccionando un nuevo tema o cambiando los colores del sitio. Para obtener más información sobre cómo personalizar el estilo, consulte Diseñar experiencias de aplicaciones móviles.
  • Control de acceso - Puede restringir el acceso a la aplicación publicada mediante el inicio de sesión único. Para obtener más información sobre el control de acceso, consulte Controlar el acceso a una experiencia.
  • Descripción general - Puede editar el nombre, la descripción y el idioma de la experiencia. Para obtener más información sobre la edición de la descripción general del sitio, consulte Detalles de la experiencia de edición.

Vista previa y publicación de la experiencia

La vista previa de una experiencia de aplicación móvil le permite ver cómo se mostrará la experiencia en un dispositivo móvil. Las experiencias se pueden obtener una vista previa en una nueva pestaña del navegador o se puede generar un enlace para compartir para que otros puedan obtener una vista previa del sitio. La publicación de la experiencia genera una clave de acceso que es requerida por el Aplicación móvil Brightcove Engage. Para obtener más información sobre cómo obtener una vista previa y publicar una experiencia, consulte Vista previa y publicación de una experiencia de aplicación móvil.
