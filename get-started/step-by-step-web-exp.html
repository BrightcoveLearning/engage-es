Paso a paso: Crear y publicar una experiencia web

Este tema proporciona un conjunto de pasos que puede seguir para crear y publicar una experiencia web con Brightcove Engage™.

Después de completar este tema, deberías poder:

  • Utilice Brightcove Engage para crear una nueva experiencia web
  • Añadir vídeos a una experiencia
  • Personalizar una experiencia
  • Vista previa y publicación de una experiencia
Nota: Para obtener una guía paso a paso para crear una experiencia de aplicación móvil, consulte Paso a paso: Creación y publicación de una experiencia de aplicación móvil.

Audiencia

Cualquiera que necesite crear portales de video web para mostrar contenido de video.

Requisitos previos

  • Una cuenta Brightcove Engage (registrarse para una prueba)

Antes de empezar

Esta guía paso a paso utilizará los videos de muestra que se incluyen con la cuenta de prueba de Brightcove Engage. Siéntase libre de subir y utilizar sus propios vídeos.

Creando una experiencia web

Para crear una experiencia web con Brightcove Engage, completará las siguientes tareas:

  • Organizar vídeos mediante listas de reproducción
  • Crea una nueva experiencia web
  • Modificar los detalles del sitio
  • Agregar imágenes y texto a una plantilla de página
  • Crea colecciones y añade vídeos a las colecciones
  • Vista previa de la experiencia
  • Cambiar el estilo del sitio
  • Publicar la experiencia

Organización de vídeos mediante listas de reproducción

Antes de empezar a crear experiencias, es útil pensar cómo quieres organizar o agrupar tus vídeos. Normalmente, querrás agrupar vídeos relacionados juntos. Por ejemplo, es posible que desee agrupar los vídeos del CEO, los vídeos de RRHH juntos, los vídeos de marketing, etc. Cuando los espectadores usen su experiencia web, verán una lista de estos grupos, seleccionarán un grupo y luego elegirán los videos que quieren ver. Cuando se utiliza Brightcove Engage, los vídeos relacionados se agrupan mediante listas de reproducción.

Nota: Si utilizas tus propios vídeos, crea varias listas de reproducción para organizar los vídeos.

Para organizar los vídeos de muestra en listas de reproducción, sigue estos pasos.

  1. Inicie sesión en Brightcove Engage. los Proyectos aparecerá la página.
  2. Haga clic en el Videos pestaña en el encabezado de navegación. Aparecerá una lista de los vídeos de ejemplo.
  3. En la navegación izquierda, haz clic en + Crear lista de reproducción.
  4. Establezca el tipo de lista de reproducción en Manual.
  5. Establezca el nombre de la lista de reproducción en Featured Video.
  6. Marca Mostrar lista de reproducción en la barra lateral.
  7. Haz clic en Guardar.
  8. Con la misma configuración, crea dos listas de reproducción más Sales Videos y HR Videos. Deberías crear tres listas de reproducción.
  9. En la navegación izquierda, haz clic en Todos los vídeos.
  10. Haga clic en la casilla de verificación junto al Reunión All Hands 2019 video y luego haga clic en Más> Agregar a la lista de reproducción.
  11. Selecciona el vídeo destacado lista de reproducción y luego haga clic en Agregar.
  12. Seleccione los siguientes videos y agréguelos al Videos de ventas lista de reproducción.
    • Reunión de estrategia de ventas 2018
    • Instrucciones de ventas - Desarrollo de su oleoducto en nuevos mercados - Ep 02
    • Instrucciones de ventas - Nuevas técnicas para las excursiones de venta superior - Ep 01
  13. Seleccione los siguientes videos y agréguelos al Videos de recursos humanos lista de reproducción.
    • RRHH - Creación de su plan de comunicaciones internas - E02
    • HR - Incorporación de nuevos empleados - E01

Todos los vídeos están ahora organizados en listas de reproducción. Al hacer clic en el nombre de una lista de reproducción en la navegación izquierda, se mostrarán los vídeos de esa lista de reproducción.

Para obtener más información sobre las listas de reproducción, consulte la Creación y gestión de listas de reproducción tema.

Creando una nueva experiencia web

Ahora que los videos están organizados en listas de reproducción, podemos crear una nueva experiencia. Para crear una nueva experiencia web, siga estos pasos.

  1. En el encabezado de navegación, haga clic en el Proyectos pestaña.
  2. Confirmar la Crear experiencia aparece la página. Si aparece una lista de experiencias, haga clic en Nueva experiencia para mostrar el Crear experiencia página.
  3. Localizar el Plantilla Web y haga clic en Escoger.
    Nota: Una vez que selecciones una plantilla, no se puede cambiar. Para obtener más información sobre las plantillas, consulte la Descripción general de las plantillas de experiencia tema.
  4. Ingresar Experiencia de muestra de plantilla web Para el Título de la experiencia. ¿Para la experiencia de pre-poblado?, selecciona No, gracias.
  5. Haz clic en Crear experiencia.

Se abrirá Experience Editor. El editor de experiencias se utiliza para personalizar la experiencia y brinda la capacidad de personalizar diseños de página, agregar videos, elegir estilos y configurar otras configuraciones. Se accede a todos los ajustes de personalización mediante el menú de navegación izquierdo.

Edición de los detalles del sitio

La configuración del título, la descripción y el idioma de la experiencia se pueden editar mediante el Editor de experiencias.

  1. Hacer clic VISIÓN DE CONJUNTO en la navegación de la izquierda.
  2. Para el Descripción de la experiencia , agregar Experiencia web de muestra.
  3. Haz clic en Guardar.

Agregar imágenes y texto a una plantilla de página

Cada plantilla web proporciona un conjunto de plantillas de página que se pueden personalizar. La plantilla web proporciona Casa , Categoría y Detalle del video plantillas de página. Estas páginas se pueden ver en el margen izquierdo como parte del Paginas menú.

Las plantillas de página mostrarán marcadores de posición en la página y un icono de lápiz () indica que el objeto se puede editar. Estos marcadores de posición se pueden utilizar para añadir imágenes, texto y enlaces a la página.

Para obtener más información sobre cómo agregar objetos a las plantillas de página, consulte Adición de imágenes, texto y enlaces a páginas de experiencia web.

Para agregar objetos a una plantilla de página, siga estos pasos.

  1. Haz clic en PÁGINAS > Inicio en la navegación izquierda. Iconos de lápiz () indican áreas de la página donde se pueden agregar imágenes / texto / enlaces. Los objetos de texto estático en la página se pueden editar haciendo clic en ellos.
  2. En la parte superior de la página, haga clic en + Agregar contenido.
  3. Haga clic en el Subir opción y luego haga clic en Ahorrar. Seleccionar Subir agregará un marcador de posición de imagen a la página.
  4. Haga clic en + Agregar foto.
  5. Hacer clic Elija el archivo y luego busque y seleccione una imagen para usar para el encabezado. Hacer clic Ahorrar.
    Nota: Brightcove redimensionará la imagen a 140 px x 50 px.
  6. Verifique que la imagen aparezca en el encabezado de la página.
  7. En la parte inferior de la página, haga clic en PLACEHOLDER en el lado izquierdo del pie de página. Ingrese el nombre de su empresa y luego haga clic en Ahorrar.
  8. Haga clic en el Categoría y Detalle del video páginas en el margen izquierdo. Observe que la imagen del encabezado y el texto del título también aparecen en esas páginas. Vuelve al Casa página.

Creación de colecciones y adición de vídeos a las colecciones

Con el Editor de experiencias, puede seleccionar qué vídeos desea mostrar en la experiencia. Al crear experiencias, los vídeos se organizan en colecciones. Para este ejemplo, las colecciones se basarán en las listas de reproducción que fueron creado antes.

Para obtener más información sobre cómo crear colecciones y añadir vídeos a una experiencia, consulte Adición de vídeos a una experiencia.

Para añadir vídeos a la experiencia, sigue estos pasos.

  1. Hacer clic VIDEOS en la navegación de la izquierda.
  2. Nombra la primera colección Ventas y luego presione ingresar.
  3. Haga clic en el nombre de la colección para abrir el editor de colecciones.
  4. Selecciona Lista de reproducción para ver los criterios.
  5. Haga clic en el Videos de ventas lista de reproducción. Los videos de esa lista de reproducción aparecerán en la lista de videos. Haz clic en Guardar.
    Nota: Cuando se utiliza una asignación de video de lista de reproducción, el orden de los videos se basa en la configuración de la lista de reproducción que se estableció en el Videos página.
  6. Crea otra colección llamada HORA utilizando el Videos de recursos humanos lista de reproducción como criterio.
  7. Confirme que se han creado 2 colecciones.
    Nota: Puede cambiar el orden de las colecciones haciendo clic y arrastrándola a una nueva posición.

Vista previa de la experiencia

La vista previa de una experiencia le permite ver cómo se mostrará el sitio en dispositivos móviles, tabletas y computadoras de escritorio. Querrá obtener una vista previa de la experiencia a medida que sigue los pasos para personalizarla. Se puede obtener una vista previa de la experiencia en una nueva pestaña del navegador o se puede generar un enlace para compartir para que otros puedan obtener una vista previa del portal. Para obtener más información sobre cómo obtener una vista previa y publicar una experiencia web, consulte Vista previa y publicación de una experiencia web.

Para obtener una vista previa de la experiencia, sigue estos pasos.

  1. Mientras edita la experiencia, haga clic en Avance y luego haga clic en Vista previa receptiva.
  2. La vista previa se abrirá en una nueva pestaña del navegador.
  3. Cierre la pestaña de vista previa del sitio.

Cambiar el estilo del sitio

La configuración de Estilo se puede utilizar para personalizar los colores y las fuentes del sitio que se utilizan en la experiencia. Para obtener más información sobre cómo cambiar el estilo del sitio, consulte Personalización del estilo de una experiencia web.

Para cambiar el estilo del sitio, siga estos pasos.

  1. Hacer clic ESTILO en la navegación de la izquierda.
  2. A Tema es un conjunto de colores y fuentes que se pueden aplicar a una plantilla. Seleccione un Tema. Este ejemplo usará el Sombras de azul tema.
  3. Seleccione un Primario y Secundario fuente a utilizar. La Título del sitio fuente controla la fuente del título utilizada en el encabezado. Si siguió este documento paso a paso, anteriormente se agregó una imagen al encabezado.
  4. Haga clic en el <Volver a la experiencia enlace en la parte superior del menú Estilo.

Publicación de la experiencia

La publicación de una experiencia web la hace disponible en una URL pública. Para obtener más información sobre cómo obtener una vista previa y publicar una experiencia de portal, consulte Vista previa y publicación de una experiencia web.

Para publicar la experiencia, sigue estos pasos.

  1. Desde dentro del Editor de experiencias, haga clic en Publicar.
  2. En el cuadro de diálogo Publicar sitio, haga clic en Publica ahora para confirmar la publicación.

Haga clic en la URL del sitio para abrirlo en otra pestaña del navegador.
