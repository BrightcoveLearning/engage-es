Descripción general: Creación, edición y publicación de experiencias web

En este tema, aprenderá los pasos básicos para crear, editar y publicar una experiencia web.

Brightcove Engage™ le permite crear experiencias de vídeo atractivas sin necesidad de recursos técnicos. Las experiencias web siguen principios de diseño receptivo para que se vean geniales en computadoras de escritorio, tabletas y dispositivos móviles.

El siguiente es un ejemplo de una experiencia web que se creó con la plantilla web Brightcove Engage.

Para obtener una guía paso a paso para crear una experiencia web, consulte Paso a paso: Crear y publicar una experiencia web.

Los pasos principales para crear y publicar una experiencia web son:

  • Elige una plantilla
  • Añadir vídeo y personalizar la experiencia
  • Vista previa y publicación de la experiencia

Elección de una plantilla

Cuando creas una nueva experiencia, empiezas por elegir una plantilla. La plantilla controla el aspecto general de la experiencia. Brightcove Engage proporciona un conjunto de plantillas que le brindan la flexibilidad de brindar el tipo de experiencia de video necesaria para interactuar con sus espectadores. Para obtener más información sobre las plantillas y ver algunos sitios de muestra, consulte la Descripción general de las plantillas de experiencia tema.

Después de elegir una plantilla, tiene la opción de rellenar previamente la experiencia con contenido de muestra. Haciendo clic Sí, rellenar previamente mi experiencia completará previamente la experiencia con videos de su cuenta. Esto hace que sea más fácil ver cómo será la experiencia al iniciar el proceso de personalización.

Adición de vídeo y personalización de la experiencia

Después de crear una experiencia, el Editor de experiencias se utiliza para personalizar la experiencia. Se pueden realizar los siguientes tipos de personalizaciones:

  • Paginas - Se pueden agregar imágenes, texto y enlaces a las plantillas de página para marcar la experiencia según sea necesario. Para obtener más información sobre cómo agregar objetos a las páginas, consulte Personalización del contenido de las páginas de experiencia web.
  • Vídeos: las colecciones se crean para seleccionar y organizar el contenido de vídeo. Para obtener más información sobre cómo agregar contenido de video al sitio, consulte Agregar videos a una experiencia.
  • Estilo - El aspecto de la experiencia se puede personalizar seleccionando un nuevo tema o cambiando los colores y fuentes del sitio. Para obtener más información sobre cómo personalizar el estilo, consulte Personalización del estilo de una experiencia web.
  • Control de acceso - Restringir el acceso al sitio. Para obtener más información sobre la configuración del sitio, consulte Controlar el acceso a una experiencia.
  • Descripción general - Edite el nombre, la descripción y el idioma de la experiencia. Para obtener más información sobre la edición de la descripción general del sitio, consulte Detalles de la experiencia de edición.

Vista previa y publicación de la experiencia

La vista previa de una experiencia le permite ver cómo se mostrará la experiencia en dispositivos móviles, tabletas y computadoras de escritorio. Las experiencias se pueden obtener una vista previa en una nueva pestaña del navegador o se puede generar un enlace para compartir para que otros puedan obtener una vista previa del sitio. La publicación de la experiencia genera una URL estática para el sitio y lo hace accesible para los espectadores. Para obtener más información sobre cómo obtener una vista previa y publicar una experiencia web, consulte Vista previa y publicación de una experiencia web.
