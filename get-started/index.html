Empezar

Brightcove Engage™ facilita la creación y publicación de experiencias de videos para que los espectadores puedan consumir su contenido donde sea que estén. Se pueden crear dos tipos de experiencias:

  • Aplicación movil - Cree una aplicación de dispositivo para que sus videos se puedan ver dentro de una aplicación nativa de iOS o Android (aprender cómo...)
  • Aplicación web - Cree un sitio web de videos para que sus videos se puedan ver en una URL específica en navegadores de escritorio y móviles (aprender cómo...)
