Requisitos del sistema Brightcove Engage

Esta página resume los requisitos de software y hardware para crear experiencias con Brightcove Engage, así como los requisitos para Brightcove Players.

Para trabajar con Brightcove Engage, su computadora debe cumplir con los siguientes requisitos.

  PC con Windows Mac
Sistema operativo

Windows 10
Windows 8
Windows 7

macOS 10.13 o posterior
Navegador[1]

Firefox - última versión
Chrome - última versión
Internet Explorer 11 o mas nuevo
Edge - última versión

Firefox - última versión
Chrome - última versión
Safari - última versión

Notas

[1] Es posible que las versiones anteriores del navegador sigan funcionando. Brightcove no prueba ni verifica la compatibilidad total con todas las versiones del navegador.
