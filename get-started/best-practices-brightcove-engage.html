Brightcove Engage™ Mejores prácticas

En este tema, aprenderá sobre las mejores prácticas recomendadas al trabajar con Brightcove Engage.

Trabajar con videos

Las cuentas de prueba de Brightcove Engage tienen las siguientes limitaciones:

  • Uso de listas de reproducción manuales para organizar videos
  • Usar esas listas de reproducción para colecciones para portales móviles / web de modo que solo necesite 1 lista de reproducción para ambos
  • Cómo actualizar fácilmente los proyectos simplemente agregando una lista de reproducción a ambos
  • Tamaños de imagen para cada tipo de proyecto
  • Guías de color para cada uno
  • La mejor manera de usar la lista de reproducción destacada para hacer el primer video en el dispositivo móvil y el primer conjunto en la web: número óptimo en la web ... Creo que dijimos 4 u 8. Eso’s arriba en el hilo.
  • comenzando con el portal móvil
